# IntimateMap Website

Site web officiel Next.js pour IntimateMap - Application de journal intime géolocalisé.

## Structure du Site

- **/** - Page d'accueil
- **/guide** - Guide de démarrage
- **/creer-un-spot** - Documentation création de spots
- **/coffre-fort** - Documentation coffre-fort sécurisé
- **/faq** - Questions fréquentes
- **/privacy** - Politique de confidentialité RGPD

## Technologies

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React 19

## Développement

```bash
npm install
npm run dev
```

Site accessible sur http://localhost:3000
